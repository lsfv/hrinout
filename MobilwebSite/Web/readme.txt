col-xs-3:
3的意思是 占一行的3/12: 最小约等于 42*3
xs意思是当前页面宽只要比xs定义的宽，我就进行分割。否则不分割，以防分割太小换行太严重，
例如当页面非常小col-xs- 都等于col-xs-12，全铺满的意思。

Bootstrap 4 网格系统有以下 5 个类:

.col- 针对所有设备
.col-xl- xiao - 屏幕宽度等于或大于 508px
.col-sm- 平板 - 屏幕宽度等于或大于 576px
.col-md- 桌面显示器 - 屏幕宽度等于或大于 768px)
.col-lg- 大桌面显示器 - 屏幕宽度等于或大于 992px)
.col-xl- 超大桌面显示器 - 屏幕宽度等于或大于 1200px)
